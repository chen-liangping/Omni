# 📌 页面关系

1. **列表页** 👉 点击「👁 详情」进入 **仓库详情页**。
2. **仓库详情页** 👉 点击 **某条部署记录** 进入 **部署日志详情页**。
3. **部署日志详情页** 👉 提供日志完整内容、下载、重试部署。
